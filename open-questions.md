# Open questions

Things that look wrong, are unresolved, or were deferred — with the
reason captured so future-you can re-evaluate.

This is the parking lot. Items here mature into:

- Roadmap entries (when the answer is "yes, do it") → `roadmap.md`
- ADRs (when the answer is "yes, with these tradeoffs") → `decisions/`
- Closed entries (when the answer is "no, here's why") — keep in this
  file under "Resolved" with date
